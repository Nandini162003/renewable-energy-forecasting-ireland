# renewable-energy-forecasting-ireland

## Project Overview
This project explores renewable energy trends in Ireland and builds forecasting models to predict future energy production patterns.

## Technologies Used
- Python
- Pandas
- Scikit-Learn
- XGBoost
- Matplotlib

##  Approach
- Exploratory Data Analysis (EDA)
- Feature engineering
- Model optimization using GridSearchCV
- Implemented Random Forest and XGBoost models

## Results
XGBoost provided improved forecasting accuracy after hyperparameter tuning.

##  Key Skills Demonstrated
Time-series forecasting, model optimization, feature engineering, data-driven insights.
